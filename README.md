# VRCBed_202004
Bed Asset for VRChat


VRCBed (Prototype)

【内容】
このアセットは、アバタを横倒す姿勢にするアセットになります。このアセットには、シーンファイルを含みませんまた、この制作にあたって、下記URLの記事の内容を参考にしました。

【Content】
